# Digitālā volejbola treniņu sistēma

## Par projektu

**Digitālā volejbola treniņu sistēma** ir tīmekļa lietojumprogramma, kas paredzēta volejbola treniņu reģistrācijas procesa modernizēšanai un vienkāršošanai. Sistēma ļauj treneriem un sportistiem efektīvi pārvaldīt treniņu grafikus, pieteikties nodarbībām un sekot apmeklējumiem.

## Funkcionalitāte

- **Treniņu kalendārs**: Treneri var izveidot un pārvaldīt treniņu laikus, redzēt pieteikušos dalībniekus un sekot izmaiņām.
- **Reģistrācijas sistēma**: Sportisti var pieteikties treniņiem, izmantojot intuitīvu tiešsaistes saskarni.
- **Apmeklējumu uzskaite**: Automātiski tiek fiksēts, cik bieži esi nācis uz treniņiem.
- **Dalībnieku redzamība**: Treneri var redzēt, cik bieži un kuri sportisti apmeklē treniņus.
